About

Beoro Admin is a HTML5/CSS3 backend template based on Twitter Bootstrap Framework, it's easily customisable, cross-browser compatible and works great on desktops, tablets and mobile devices. It can be used for any web aplication (Cms, Crm, Statistics, Shop backend etc).

Features

  • Responsive Layout (desktops, tablets, mobile devices)
  • HTML5/CSS3
  • Based on Twitter Bootstrap Framework
  • Custom pages: Blog, Error 404, Help/Faq, Invoices, Mailbox
  • Integrated plotting libray (bars, lines, pies, donuts )
  • Sticky Footer
  • Form Elements
    • Sliders, Progressbars
    • 2 column multiselect
    • Enchanced select boxes
    • Masked inputs
    • Passwors strength meter
    • Datepicker
    • Timepicker
    • Colorpicker
    • Switch buttons
    • Autosize textareas
    • Textarea character/word counter
    • UI Spinners
    • WYSIWG Editor
    • Drag∧Drop Multiupload
  • Form Validation
  • Responsive gallery grid with basic edit options
  • jQuery Calendar
  • Contact List
  • Datatables
  • File manager
  • Google maps
  • Sticky notifications and extended bootstrap modals
  • Tooltips and Popovers
  • Widget Boxes
  • Included three Icon Sets
  • jQuery Wizard
